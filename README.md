# EDS Radar 📡

Türkiye'deki EDS (Elektronik Denetleme Sistemi) hız kameralarını haritada gösteren ve yaklaşırken uyarı veren ücretsiz uygulama.

## Özellikler

- 🗺️ 1572 EDS kamerası Google Haritalar üzerinde
- 🔔 Yaklaşınca sesli + bildirim uyarısı
- 📍 Gerçek zamanlı GPS takibi
- 🛣️ OHTS (otoyol) ortalama hız bölgesi takibi
- 🔍 Kamera tipi filtreleme (PTS-EDS, Belediye, Otoyol)
- 📋 Kamera listesi ve arama
- ⚡ Arka planda çalışma (ekran kapalıyken)

## İndir

### Android
👉 [En son sürümü indir (APK)](../../releases/latest)

> **Kurulum:** APK indirildikten sonra "Bilinmeyen kaynaklardan yükle" iznini ver.
> Ayarlar → Güvenlik → Bilinmeyen uygulamalar → Tarayıcı → İzin ver

### iOS (iPhone)
🍎 App Store'da yayın hazırlığı devam ediyor.

## Kullanım

1. Uygulamayı aç
2. Konum izni ver ("Her Zaman" seç - arka plan takibi için)
3. **Takip** butonuna bas
4. Harita konumuna zoom yaparak takip başlar
5. Kameraya yaklaşınca otomatik uyarı gelir

## Kamera Tipleri

| Tip | Açıklama |
|-----|----------|
| PTS-EDS | Polis Trafik Sistemi elektronik denetleme |
| Bld | Belediye kamerası |
| Bld-EDS | Belediye elektronik denetleme |
| Otoyol | Karayolları otoyol kamerası |

## Sorumluluk Reddi

Bu uygulama yalnızca bilgilendirme amaçlıdır. Trafik kurallarına uymak sürücünün sorumluluğundadır. Uygulama kullanımından doğabilecek herhangi bir ihlal veya kaza için geliştirici sorumlu tutulamaz.
